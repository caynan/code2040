$ code2040_

  code2040 is a simple CLI for using the CODE2040 API
  it could be used to:
    - get a token
    - get & post solution to problems
    - get status

  Requires Click 2.0 or higher.

Usage:

  The `--editable` tag is optional, use when modifing the script
  $ pip install --editable .
  $ code2040 --help
